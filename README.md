# 📍 Kutaisi Community Map (ქუთაისის რუკა)

ეს არის ინტერაქტიული რუკა ქუთაისისთვის, სადაც მოქალაქეებს შეუძლიათ დააფიქსირონ სხვადასხვა პრობლემები ან მოვლენები.

### 🚀 ფუნქციები (Features):

- **ადგილმდებარეობის დადგენა:** GPS-ით საკუთარი მდებარეობის პოვნა.
- **რეპორტინგი:** ფოტოს ან ვიდეოს მიმაგრება პირდაპირ ტელეფონიდან.
- **ფილტრაცია:** კატეგორიების და სტატუსების მიხედვით (მაგ. პოლიცია, გზები, ინფრასტრუქტურა).
- **სტატუსის კონტროლი:** რეპორტის სტატუსის შეცვლა (🔴 მიმდინარეობს -> 🟢 მოგვარდა).

### 🛠️ ტექნოლოგიები:

- Leaflet.js (Map Engine)
- OpenStreetMap
- LocalStorage (მონაცემების შენახვა ბრაუზერში)
